## Market clusters

### London AI agent and developer platform roles

The strongest cluster for immediate applications is London-based or UK-remote
AI/devtools/platform work where the role needs someone who can build demos,
explain agent workflows, talk to senior engineers, and turn field feedback into
product direction.

Strong candidates: Sierra, Anthropic, Ably, Mistral, Tailscale, and Snyk if
the role is confirmed on the canonical company source.

### Berlin and European AI infrastructure roles

The best Berlin/European match is not tiny generic startup work. It is
engineering-heavy AI infrastructure and observability where a technical DevRel
person can speak credibly about agents, evals, traces, LLM workflows, and
developer adoption.

Strong candidates: Langfuse, Temporal, Qdrant if the EMEA DevRel listing is
still live, Mistral, Cohere/adjacent developer-experience engineering.

### Global remote developer-facing roles

Several global or US-heavy roles are attractive by topic but weaker by
geography. These are useful for warm intros, monitoring, or selective
applications only if the posting explicitly supports UK/EU candidates.

Examples: Runway, DataHub/Acryl, Metabase, Modal, Databricks/Neon, Fastly,
Datadog, ElevenLabs.

### Open-source database and data infrastructure growth

There is a credible adjacent cluster around open-source adoption, developer
community, technical content, and enterprise pipeline for databases/data
platforms.

Examples: QuestDB, Metabase, DataHub/Acryl, Neon, JFrog.
