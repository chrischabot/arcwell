# Wiki Ingest

The user invoked this command with: $ARGUMENTS

Ingest the supplied path or URL into the local wiki. Use `wiki_ingest_dir` for directories, `wiki_ingest_file` for local Markdown files, `wiki_ingest_job` when the user specifically wants a recorded file job, and `wiki_ingest_url` for public HTTPS URLs. Report the resulting page id or ingest report.
