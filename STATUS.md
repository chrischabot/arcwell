# Arcwell Status Matrix

Last audited: 2026-06-23

This file is the project compass and todo list. It is intentionally blunt. A package
directory, README, command prompt, or MCP tool does not mean the capability is
finished. The status below is based on the current repository, not the product
vision.

## Legend

| Status | Meaning |
| --- | --- |
| Done | Implemented, documented, and covered by relevant tests for the current intended scope. |
| Partial | Usable slice exists, but important behavior is missing or shallow. |
| Scaffold | Boundary, docs, commands, or placeholder flow exists; product behavior is not real enough yet. |
| Missing | Not implemented in any meaningful form. |
| Risk | Exists but can mislead, lose data, leak trust boundaries, or fail silently. |
